# Roadmap

## Current Phase

The project is in the prototype stage.

Implemented or largely implemented areas include:

- backend schema and models
- event store foundation
- trust and authentication foundation
- domain services
- REST API layer
- entropy-centered demo seed dataset

Areas still being actively refined include:

- CLI polish and end-to-end workflows
- frontend integration and UX completeness
- full stabilization of all test suites
- proposal and agent workflow hardening

## Near-Term Goals

1. stabilize backend and integration tests
2. finish frontend and CLI workflows around proposals, reviews, and search
3. improve demo experience with clearer seeded navigation paths
4. document reproducible evaluation scenarios for reviewers and users

## Medium-Term Goals

1. strengthen search and graph visualization
2. improve AI-assisted linking quality and reviewability
3. support richer evidence modeling and provenance tracking
4. formalize deployment and operational guidance
